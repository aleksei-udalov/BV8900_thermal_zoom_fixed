MyFLIR가 일정 간격으로 여러 장의 사진을 촬영하도록 한 후, 해당 이미지들을 사용해 비디오 클립을 만들면 타임랩스 비디오 제작이 가능합니다.

다음 순서를 따릅니다:

  1. 화면 하단의 모드 버튼을 누르면 모드 셀렉터 메뉴가 표시됩니다.
  2. 아이콘을 누르면 타임랩스 모드로 변경됩니다.
  3. (선택적 단계) 아이콘을 누르고 다른 팔레트를 선택합니다.
  4. 버튼을 누르면 타임랩스 설정 메뉴를 열 수 있습니다.
  5. 설정을 변경합니다. 변경 가능한 설정으로는 최초 딜레이(초), 프레임 간격(초), 재생 비율(초당 프레임) 및 경과 시간 표시가 있습니다.
  6. 버튼을 누르면 녹화를 시작합니다.
  7. 버튼을 누르면 녹화를 종료합니다.
