A funcionalidade de medição de zona de interesse (ROI) está disponível em ambos os modos Editar e Live. Para cada ROI é apresentada a respetiva temperatura máx./mín./média.

Estão disponíveis dois tamanhos de ROI – um ROI de Fotograma Completo e um ROI de Quadrado Central. Se mantiver premido um ROI, serão apresentadas opções para o eliminar ou para apresentar as temperaturas mínima e máxima.

Toque no botão na barra de ferramentas superior e faça um dos seguintes:

ROI de Medição Pontual

  • Toque no botão para criar uma Medição Pontual individual.
  • Arraste o ícone de Medição Pontual gerado ao longo do ecrã para medir a temperatura de uma Medição Pontual individual Pode adicionar até três medições pontuais por imagem.
  • Mantenha premidos ícones individuais de medição pontual para ver opções para eliminar o ROI

ROI de Fotograma Completo

  • Toque no botão para mostrar a temperatura máx./mín./média para a cena completa.
  • Mantenha premido qualquer ponto da cena para revelar opções para eliminar o ROI ou para apresentar as temperaturas mínima e máxima.

ROI de Quadrado Central

  • Toque no ícone mais pequeno para mostrar a temperatura máx./mín./média para o centro do fotograma.
  • O fotograma pode ser redimensionado arrastando desde qualquer canto, mas permanecerá centrado.
  • Mantenha premido o centro do fotograma para revelar opções para eliminar o ROI ou para apresentar as temperaturas mínima e máxima.
