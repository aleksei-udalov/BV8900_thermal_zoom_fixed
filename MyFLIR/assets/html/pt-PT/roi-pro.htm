A funcionalidade de medição de zona de interesse (ROI) está disponível em ambos os modos Editar e Live. Para cada ROI é possível apresentar a temperatura máx./mín./média do ROI, juntamente com marcadores da respetiva localização.

Estão disponíveis cinco tipos de ROI: Medição Pontual, Retângulo (Fotograma Completo e Móvel), Círculo e Linha. Se mantiver premido um ROI, serão apresentadas opções para o eliminar ou para apresentar as temperaturas mínima e máxima.

Toque no botão na barra de ferramentas superior e faça um dos seguintes:

Medição Pontual

  • Toque no botão para criar uma Medição Pontual individual.
  • Arraste o ícone de Medição Pontual gerado ao longo do ecrã para medir a temperatura de uma Medição Pontual individual Pode adicionar um máximo de cinco medições pontuais a uma imagem.

Fotograma Completo

  • Toque no botão para mostrar a temperatura máx./mín./média para a cena completa.

Retângulo

  • Toque no botão mais pequeno para apresentar a temperatura máx./mín./média de uma área retangular do fotograma.
  • É possível redimensionar o retângulo, arrastando a partir de qualquer canto, ou movê-lo, arrastando o respetivo centro.

Círculo

  • Toque no botão para apresentar a temperatura máx./mín./média de uma área retangular do fotograma.
  • É possível redimensionar o círculo, arrastando a partir da borda, ou movê-lo, arrastando o respetivo centro.

Linha

  • Toque no ícone para apresentar a temperatura máx./mín./média ao longo de uma linha do fotograma.
  • Para alternar entre vertical e horizontal, toque duas vezes na linha.
  • É possível mover a linha, arrantando-a.
