Pode alterar várias definições relacionadas com o comportamento de MyFLIR.

Efetue o seguinte procedimento:

  1. Toque no ícone para mostrar o menu de ferramentas e definições.
  2. Toque no ícone para mostrar o menu Definições.
  3. No menu Definições, pode fazer o seguinte:
    • Ver ficheiros de ajuda e informação acerca de MyFLIR.
    • Emissividade: Para alterar a emissividade, toque Emissividade e selecione um dos seguintes:
      • Mate (Recomendado).
      • Semi Mtte.
      • Semi Brilhante.
      • Brilhante (Não recomendado).
    • Unidade de temperatura: Para alterar a temperatura da unidade, toque em °C ou °F.
    • Guardar a localização: Para guardar a localização GPS na imagem, deslize o controlo deslizante para Ligada.
    • Calibragem automática: Para calibrar imagens automaticamente, deslize o controlo deslizante para Ligada.

O que é a emissividade?

A emissividade é a capacidade de emissão de luz infravermelha de um material. Para obter uma medição exata, a emissividade necessita corresponder ao material que está a visualizar. Para a maioria dos materiais, a definição por omissão Mate é uma boa aproximação.

Materiais brilhantes, como metais, podem fornecer medições de temperatura incorretas.
