No alinhamento das alterações das câmaras termais e visíveis, pode realizar ajustes manuais.

Processo de Realinhamento

  1. Para definir o alinhamento, toque no ícone . Depois toque no ícone e selecione Ferramenta de Realinhamento de Imagens.
  2. Coloque um objeto, por exemplo a sua mão, no campo de visão a meio metro de distância.
  3. Toque no botão de câmara para congelar a imagem e permitir os controlos de alinhamento.
  4. Arraste o dedo ao longo da imagem para ajustar o alinhamento para cima e para baixo. Use o controlo deslizante para rodar.
  5. Se o objeto que está a usar para alinhar as imagens não estiver a meio metro de distância, configure a distância com o controlo deslizante inferior.
  6. Toque em GUARDAR para aplicar as alterações.

Reverter

  • Toque em REVERTER para carregar o alinhamento de fábrica.

Cancelar

  • Toque em CANCELAR para sair da Ferramenta de Realinhamento de Imagens sem guardar.
