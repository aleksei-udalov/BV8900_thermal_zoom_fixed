Gör såhär:

  1. Tryck på knappen LÄGE i nederkanten av skärmen för att visa lägesväljarmenyn.
  2. Gör ett av följande:
    • Tryck på -ikonen för att gå till fotoläget.
    • Tryck på -ikonen för att gå till videoläget.
    • Tryck på -ikonen för att gå till livestream-läge.
    • Tryck på -ikonen för att gå till timelapse-läget.
