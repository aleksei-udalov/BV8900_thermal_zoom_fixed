Observera: detta avsnitt gäller enbart Foto- och Video-lägena.

Du kan ställa in MyFLIR på att spara en bild efter ett angivet intervall.

Gör såhär:

  1. Tryck på -ikonen för att visa menyn för verktyg och inställningar.
  2. Tryck på -ikonen.
  3. Tryck igen för att välja det önskade tidsintervallet (3 eller 10 sekunder).
  4. Tryck på - eller -knappen (i foto- respektive videoläge) för att starta timern.
