Du kan ändra en mängd inställningar för att styra hur MyFLIR fungerar.

Gör såhär:

  1. Tryck på -ikonen för att visa menyn för verktyg och inställningar.
  2. Tryck på -ikonen för att visa menyn Inställningar.
  3. På menyn Inställningarkan du göra följande:
    • Se hjälpfiler och information om MyFLIR.
    • Emissivitet: För att ändra emissiviteten, tryck på Emissivitet och välj en av följande:
      • Matt (Rekommenderat).
      • Halvmatt.
      • Halvblank.
      • Blank (Rekommenderas ej).
    • Temperaturenhet: För att ändra temperaturenhet, tryck på °C eller °F.
    • Spara plats: För att spara GPS-platsen i bilderna, svep slidern till På.
    • Automatisk kalibrering: För att automatiskt kalibrera bilder, svep slidern till På.

Vad är emissivitet?

Emissiviteten anger ett materials förmåga att skicka ut infrarött ljus. För att få en korrekt mätning måste inställningen stämma överens med emissiviteten för det material du undersöker. För de flesta material är det förinställda valet Matt en bra approximation.

Blanka material, såsom metaller, kan ge felaktiga temperaturmätningar.
