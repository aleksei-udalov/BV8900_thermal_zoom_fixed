Gör såhär:

  1. Tryck på knappen LÄGE i nederkanten av skärmen för att visa lägesväljarmenyn.
  2. Tryck på -ikonen för att gå till YouTube Live-läge.
  3. Logga in på eller välj ditt Google konto och godkänn att MyFLIR får skapa live-videor.
  4. Rikta din kamera mot det du är intresserad av.
  5. Tryck på -knappen för att starta livesändningen, vänta på att knappen indikerar att sändningen har startat:
  6. Tryck på -knappen för att dela livesändningen.
  7. Tryck på -knappen för att avsluta livesändningen.
