När du sparar en bild så sparas den i MyFLIR:s bildgalleri. Du kan när som helst öppna en bild och lägga till en spot-mätare, ändra paletten m.m.

Tryck på miniatyrbilden i skärmens nedre högra hörn för att öppna den senast tagna bilden.

Bildvisning

I bildvisning kan du göra följande:

  • Tryck på -ikonen för att gå till Biblioteksvyn.
  • Svep åt vänster/höger för att gå till tidigare/nästa bild/video.
  • (Endast bilder.) Svep upp/ner för att byta mellan värmebilder och bilder av synligt ljus.
  • (Endast bilder.) Tryck på -ikonen för att visa en vy där du kan lägga till en spot-mätare och ändra palett.
  • (Endast bilder.) Tryck på -ikonen för att visa information om bilden och, om så önskas, GPS-information om den plats där bilden togs.
  • Tryck på -ikonen för att dela bilden/videon med exempelvis e-post, Facebook eller Twitter.
  • Tryck på -ikonen för att radera bilden/videon.

Biblioteksvy

I Biblioteksvyn kan du göra följande:

  • Tryck på -ikonen för att gå till Bildvisning.
  • Tryck på en miniatyrbild för att visa den fullständiga bilden/videon.
  • Tryck och håll ner på en miniatyrbild för att gå till Val-läget där du kan göra följande:
    • Tryck på miniatyrbilderna för att välja/avvälja bilder/videor.
    • Tryck på -ikonen för att välja alla bilder/videor.
    • Tryck på -ikonen för att dela de valda bilderna/videorna via exempelvis e-post, Facebook eller Twitter.
    • Tryck på -ikonen för att radera de valda bilderna/videorna.
