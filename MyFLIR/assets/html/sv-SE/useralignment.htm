Om justeringen mellan värmekameran och kameran för synligt ljus ändras så kan du justera den manuellt.

Justeringsprocedur

  1. Tryck på -ikonen för att ställa in justeringen. Tryck sedan på -ikonen och välj Bildjusteringsverktyget.
  2. Placera ett föremål – exempelvis din hand – i synfältet, omkring en halv meter bort.
  3. Tryck på kameraknappen för att frysa bilden och aktivera justeringskontrollerna.
  4. Dra ditt finger längs bilden för att ställa in justeringen uppåt och nedåt. Använd slidern för att rotera den.
  5. Om föremålet du använder för att justera bilderna inte är en halv meter bort, kan du ange avståndet med slidern nertill.
  6. Tryck på SPARA för spara att dina ändringar.

Återgå

  • Tryck på ÅTERGÅ för att läsa in fabriksjusteringen.

Avbryt

  • Tryck på AVBRYT för att avsluta Bildjusteringsverktyget utan att spara.
