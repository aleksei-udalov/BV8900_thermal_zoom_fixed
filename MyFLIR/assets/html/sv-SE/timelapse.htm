Du kan skapa en timelapse-video genom att låta MyFLIR ta flera bilder under ett inställt tidsintervall och därefter skapa ett videoklipp med hjälp av bilderna.

Gör såhär:

  1. Tryck på lägesväljarknappen nederst på skärmen för att visa menyn för lägesväljaren.
  2. Tryck på -ikonen för att aktivera TIMELAPSE-läget.
  3. (Valfritt steg.) Tryck på -ikonen och välj en annan palett.
  4. Tryck på -knappen för att öppna menyn för timelapse-inställningar.
  5. Gör inställningarna Inställningarna du kan göra är Inledande fördröjning (sekunder), Bildintervall (sekunder), Uppspelningshastighet (bilder per sekund) och Visa förfluten tid.
  6. Tryck på -knappen för att starta inspelningen.
  7. Tryck på -knappen för att stoppa inspelningen.
