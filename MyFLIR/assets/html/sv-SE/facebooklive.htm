Gör såhär:

  1. Tryck på knappen LÄGE i nederkanten av skärmen för att visa lägesväljarmenyn.
  2. Tryck på -ikonen för att gå till Facebook Live-läge.
  3. Logga in till ditt Facebook-konto och godkänn att MyFLIR postar livevideor.
  4. Rikta din kamera mot det du är intresserad av.
  5. Välj i dropdown-menyn intill avtryckaren för vilken publik du vill visa din live-video:
    • offentligt:
    • vänner:
  6. Tryck på -knappen för att starta livestreamen.
  7. Tryck på -knappen för att stoppa livestreamen.
