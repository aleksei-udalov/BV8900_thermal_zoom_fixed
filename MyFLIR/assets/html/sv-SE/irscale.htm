IR-skalan låter dig styra hur färgerna i paletten appliceras på olika temperaturer i bilden. Som förinställt val anpassar sig IR-skalan automatiskt.

  • För att visa IR-skalan, tryck på -omkopplaren.
  • Temperaturintervallet kan låsas/låsas upp genom att trycka på -omkopplaren.
  • När man dubbeltrycker på IR-skalan återställs de övre/nedre temperaturgränserna till de automatiskt förvalda värdena.
  • Dra i skalan för att justera intervallet.
  • Tryck på sifferrutan för att aktivera direkt inmatning av talet.
