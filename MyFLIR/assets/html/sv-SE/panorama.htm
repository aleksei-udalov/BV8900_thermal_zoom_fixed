I telefoner som har stöd för accelerometer och gyroskop kan du använda panoramafunktionaliteten.

Du kan skapa en panoramabild genom att vrida kameran från vänster till höger eller från höger till vänster.

Gör såhär:

  1. Tryck på knappen LÄGE i nederkanten av skärmen för att visa lägesväljarmenyn.
  2. Tryck -ikonen för att aktivera panoramaläge.
  3. (Valfritt steg.) Tryck på -ikonen och välj en annan palett.
  4. Tryck på -knappen för att starta panoreringssekvensen.
  5. Hål kamera stadigt och panorera från vänster till höger eller från höger till vänster. Appen detekterar automatiskt riktningen och visar hjälpande grafik.

  6. Tryck på -knappen för att avsluta sekvensen. Om det gick bra att skapa panoramat så sparas det automatiskt.
