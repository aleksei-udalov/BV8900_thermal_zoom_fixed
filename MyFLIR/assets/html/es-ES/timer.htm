Nota: Esta sección solo hace referencia a los modos de foto y vídeo.

Puede configurar MyFLIR para que guarde una imagen después de un intervalo especificado.

Siga este procedimiento:

  1. Pulse el icono para mostrar las herramientas y el menú de ajustes.
  2. Pulse el icono .
  3. Vuelva a pulsarlo para seleccionar el intervalo de tiempo deseado (3 o 10 segundos).
  4. Pulse el botón o (en modo foto o modo vídeo, respectivamente) para iniciar el temporizador.
