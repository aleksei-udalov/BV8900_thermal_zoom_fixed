Por defecto, la imagen se calibra de forma automática para garantizar una imagen optimizada y mostrar una temperatura correcta. Mientras se realiza el calibrado, la imagen se congela y el botón muestra el progreso.

Puede cambiar entre los modos de calibrado automático y manual. En el modo manual, tendrá que calibrar la imagen de forma manual.

  • Para calibrar la imagen de forma manual, pulse el botón .
  • Para cambiar entre el modo automático y manual, pulse el botón . Pulse el icono y cambie el ajuste debajo de Calibrado automático.
