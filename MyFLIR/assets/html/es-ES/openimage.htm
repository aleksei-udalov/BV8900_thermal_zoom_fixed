Cuando guarde una imagen, esta se almacenará en la galería de imágenes de MyFLIR. Puede abrir una imagen en cualquier momento y añadir un medidor de punto, cambiar la paleta, etc.

Para abrir la última imagen tomada, pulse la foto en miniatura de la esquina inferior derecha de la pantalla.

Vista Imagen

En la vista Imagen, puede realizar las acciones siguientes:

  • Pulse el icono para ir a la fototeca.
  • Deslice hacia la izquierda/derecha para ir a la imagen/vídeo anterior/siguiente.
  • (solo imágenes). Deslice hacia arriba/abajo para alternar entre imágenes térmicas y visuales.
  • (solo imágenes). Pulse el icono para mostrar una ventana donde podrá añadir un medidor de punto y cambiar la paleta.
  • (solo imágenes). Pulse el icono para mostrar información sobre la imagen y, de forma opcional, sobre la ubicación GPS desde donde se hizo.
  • Pulse el icono para compartir la imagen o el vídeo utilizando, por ejemplo, el correo electrónico, Facebook o Twitter.
  • Pulse el icono para eliminar la imagen o el vídeo.

Vista Fototeca

En la vista Fototeca, puede realizar las acciones siguientes:

  • Pulse el icono para ir a la vista Imagen.
  • Pulse una miniatura para ver la imagen o el vídeo completo.
  • Mantenga pulsada la miniatura para ir al modo selección, donde podrá realizar lo siguiente:
    • Pulsar las miniaturas para seleccionar o deseleccionar imágenes o vídeos.
    • Pulsar el icono para seleccionar todos los vídeos o imágenes.
    • Pulsar el icono para compartir los vídeos o imágenes seleccionados mediante, por ejemplo, el correo electrónico, Facebook o Twitter.
    • Pulsar el icono para eliminar los vídeos o imágenes seleccionados.
