Puede cambiar una gran variedad de ajustes sobre el comportamiento de MyFLIR.

Siga este procedimiento:

  1. Pulse el icono para mostrar las herramientas y el menú de ajustes.
  2. Pulse el icono para mostrar el menú de ajustes.
  3. En el menú de ajustes, puede realizar lo siguiente:
    • Ver archivos de ayuda e información acerca de MyFLIR.
    • Emisividad: para cambiar la emisividad, pulse Emisividad y seleccione una de las siguientes opciones:
      • Mate (recomendado).
      • Semimate.
      • Semibrillante.
      • Brillante (no recomendado).
    • Unidad de temperatura: para cambiar la unidad de temperatura, pulse °C o °F.
    • Guardar ubicación: para guardar la ubicación GPS en la imagen, mueva la barra deslizante a la posición Sí.
    • Calibrado automático: para calibrar las imágenes de forma automática, mueva la barra deslizante a la posición Sí.

¿Qué es la emisividad?

La emisividad es la capacidad que tiene un material para emitir radiación infrarroja. Para obtener una medida precisa, la emisividad tiene que coincidir con el material que está visualizando. Para la mayoría de materiales, el ajuste Mate por defecto es una buena aproximación.

Los materiales brillantes, como los metales, pueden producir medidas de temperatura incorrectas.
