Suivez cette procédure :

  1. Tapez sur le bouton MODE en bas de l'écran pour afficher le menu de sélection de mode.
  2. Tapez sur l'icône pour accéder au mode vidéo.
  3. Pointez la caméra vers votre sujet d'intérêt.
  4. Tapez sur le bouton pour démarrer l'enregistrement.
  5. Tapez sur le bouton pour arrêter l'enregistrement.
