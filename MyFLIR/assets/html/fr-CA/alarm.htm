Activer les alarmes déclenchées en entrant le mode isotherme et en cliquant sur le bouton d’alarme dans le menu des options d'isotherme. Les alarmes ont trois options pour déclencher des événements par température quand le bouton d'alarme est poussé:

  • Capturer photo
  • Vibrer
  • Pousser notification
La période d'attente entre chaque alarme successive est de 5 secondes. Le bouton d'alarme sera affiché comme quand l'alarme est activée.
