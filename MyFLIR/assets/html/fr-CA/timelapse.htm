Vous pouvez créer une vidéo accélérée en laissant MyFLIR capter quelques images dans un intervalle de temps réglé et ensuite créer un vidéoclip basé sur ces images.

Suivez cette procédure :

  1. Tapez sur le bouton MODE dans le bas de l'écran pour afficher le menu de sélection de mode.
  2. Tapez sur l'icône pour accéder au mode ACCÉLÉRÉ.
  3. (Étape facultative) Tapez sur l'icône et sélectionnez une palette différente.
  4. Tapez sur le bouton pour ouvrir les réglages du menu accéléré.
  5. Définir les réglages Les réglages qui peuvent être définis sont Délai initial (secondes), Intervalle de trame (secondes), Cadence de lecture (trames par seconde), et Affichage du temps écoulé.
  6. Tapez sur le bouton pour démarrer l'enregistrement.
  7. Tapez sur le bouton pour arrêter l'enregistrement.
