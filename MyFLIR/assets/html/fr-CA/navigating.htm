Suivez cette procédure :

  1. Tapez sur le bouton MODE en bas de l'écran pour afficher le menu de sélection de mode.
  2. Faire un des suivants:
    • Taper l'icône pour activer le mode Photo.
    • Taper l'icône pour activer le mode Vidéo.
    • Taper l'icône pour activer le mode Diffusion en direct.
    • Taper l'icône pour activer le mode intervalle régulier.
