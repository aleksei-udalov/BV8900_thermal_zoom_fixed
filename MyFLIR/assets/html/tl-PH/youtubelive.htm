Sundin ang pamamaraang ito:

  1. I-tap ang button na MODE sa ibaba ng screen para ma-display ang menu ng tagapili ng mode.
  2. I-tap ang icon na para mapunta sa YouTube Live mode.
  3. Mag-log in sa o piliin ang inyong Google account at hayaan ang MyFLIR na gumawa ng mga live video
  4. Ituro ang kamera patungo sa paksang pinag-kakainteresan mo.
  5. I-tap ang button na para umpisahan ang live stream, maghintay na tukuyin ng button na nagsimula na ang stream:
  6. I-tap ang button na para i-share ang live stream.
  7. I-tap ang button na para ihinto ang live stream.
