Maaari mong gamitin ang app para makontrol ang built-in na ilaw. Ang paggamit ng ilaw na ito ay magpapahusay sa mga detalye ng dulo ng mga bagay sa eksena.

Sundin ang pamamaraang ito:

  1. I-tap ang icon para ma-display ang mga tool at menu ng mga setting.
  2. I-tap ang icon para buksan ang ilaw.
