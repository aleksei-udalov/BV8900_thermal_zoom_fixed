Sundin ang pamamaraang ito:

  1. I-tap ang button na MODE sa ibaba ng screen para ma-display ang menu ng tagapili ng mode.
  2. I-tap ang icon para pumunta sa Video mode.
  3. Ituro ang kamera patungo sa paksang pinag-kakainteresan mo.
  4. I-tap ang button para simulan ang pag-record.
  5. I-tap ang button para ihinto ang pag-record.
