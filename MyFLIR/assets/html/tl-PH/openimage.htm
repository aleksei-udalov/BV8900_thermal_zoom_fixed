Kapag nag-save ka ng imahe, nasi-save ito sa gallery ng imahe ng MyFLIR. Sa anumang oras ay maaari mong buksan ang isang imahe at magdagdag ng spotmeter, palitan ang palette, atbp.

Para buksan ang huling kinuha na larawan, i-tap ang thumbnail ng imahe sa kanang bahagi sa ibaba ng screen.

View ng imahe

Sa view ng Imahe, maaari mong gawin ang sumusunod:

  • I-tap ang icon para makapunta sa view ng Aklatan.
  • Mag-swipe sa kaliwa/kanan para pumunta sa nauna/kasunod na imahe/video.
  • (Mga imahe lamang.) Mag-swipe pataas/pababa para lumipat sa pagitan ng thermal at mga biswal na imahe.
  • (Mga imahe lamang.) I-tap ang icon para ma-display ang isang view kung saan maaari mong idagdag ang isang spotmeter at palitan ang palette.
  • (Mga imahe lamang.) I-tap ang icon para ma-display ang impormasyon tungkol sa imahe, at opsyonal, ang lokasyon ng GPS kung saan kinuha ang imahe.
  • I-tap ang icon para i-share ang imahe/video gamit ang. hal., e-mail, Facebook, o Twitter.
  • I-tap ang icon para tanggalin ang imahe/video.

View ng aklatan

Sa view ng Aklatan, maaari mong gawin ang isa sa sumusunod:

  • I-tap ang icon para pumunta sa view ng Imahe.
  • I-tap ang thumbnail para i-display ang buong imahe/video.
  • I-tap ang i-hold ang thumbnail para pumunta sa mode ng Seleksyon, kung saan magagawa mo ang sumusunod:
    • I-tap ang mga thumbnail para piliin/tanggalin sa pagkakapili ang mga imahe/video.
    • I-tap ang icon para piliin ang lahat ng imahe/video.
    • I-tap ang icon para ibahagi ang napiling mga imahe/video gamit ang, hal., e-mail, Facebook, o Twitter.
    • I-tap ang iconpara tanggalin ang napiling mga imahe/video.
