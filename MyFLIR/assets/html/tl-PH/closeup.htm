Sa lahat ng mode ng camera, maaari kang kumuha ng close-up na litrato/video ng mga bagay sa distansyang 0.3–1 m (12–40 in. ).

Sundin ang pamamaraang ito:

  1. Iposisyon ang kamera 0.3–1 m (12–40 in.) mula sa bagay.
  2. I-tap ang button sa itaas na toolbar para ipakita ang pagpili ng fusion mode na drop-down menu.
  3. Gamitin ang slider para i-adjust ang mga outline hanggang maglilinya ang mga dulo ng infrared na bagay.
  4. I-tap ang button na capture.
