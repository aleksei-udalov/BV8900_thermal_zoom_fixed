Maaari kang gumawa ng isang time-lapse video sa pamamagitan ng pagpapahintulot sa MyFLIR na mag-capture ng ilang imahe sa isang itinakdang oras at likhain ang video clip para sa mga imaheng iyon.

Sundin ang pamamaraang ito:

  1. I-tap ang button na MODE sa ibaba ng screen para i-display ang menu ng mode selector.
  2. I-tap ang icon para pumunta sa TIME-LAPSE mode.
  3. (Opsyonal na hakbang.) I-tap ang icon at pumili ng naiibang palette.
  4. I=tap ang button para mabusan ang Time-Lapse settings menu.
  5. Ilarawan ang mga setting. Ang setting na maaaring ilarawan ay ang Unang pagkaantala (segundo), Frame Interval (seconds), Playback Rate (mga frame bawat segundo), at Ipakita ang Lumipas na Oras.
  6. I-tap ang button para magsimula sa pagrekord.
  7. I-tap ang button para ihinto ang pagrekord.
