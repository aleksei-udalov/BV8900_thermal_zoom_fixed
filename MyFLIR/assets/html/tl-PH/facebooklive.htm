Sundin ang pamamaraang ito:

  1. I-tap ang button na MODE sa ibaba ng screen para ma-display ang menu ng tagapili ng mode.
  2. I-tap ang icon para pumunta sa Facebook Live mode.
  3. Mag-log in gamit ang iyong Facebook account at payagan ang MyFLIR na mag-post ng mga live na video
  4. Ituro ang kamera patungo sa paksang pinag-kakainteresan mo.
  5. Gamitin ang pagpili ng audience na drop down na katabi ng shutter button para piliin kung sino ang makakakita ng live na video:
    • publiko:
    • mga kaibigan:
  6. I-tap ang button para simulan ang live stream.
  7. I-tap ang button para ihinto ang live stream.
