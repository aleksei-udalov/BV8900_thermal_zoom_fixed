Pinahihintulutan ka ng IR Scale na kontrolin kung paano nilalapat ang mga kulay ng palette sa mga temperatura sa imahe. Awtomatikong nag-a-adjust ang IR Scale bilang default.

  • Para ipakita ang IR Scale, i-tap ang toggle na button.
  • Ang saklaw ng temperatura ay maaaring i-lock/i-unlock sa pamamagitan ng pag-tap sa toggle button.
  • Kapag i-double-tap ang IR scale, ang mga limitasyon ng itaas/ibabang temperatura ay mai-reset sa default awtomatikong values.
  • I-drag ang scale bar para ayusin ang range.
  • I-tap ang kahon ng numero para paganahin ang direktang pag-edit ng numero.
