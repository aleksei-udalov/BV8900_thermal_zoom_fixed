Kung magbago ang pagkakapantay ng thermal at nakikitang camera, makakagawa ka ng mga mano-manong pag-aayos.

Pamamaraan sa Realignment

  1. Para i-set ang alignment, i-tap ang na icon. Pagkatapos ay i-tap ang icon at piliin ang Utility ng Pagre-align ng Imahe.
  2. Maglagay ng bagay na ang kamay mo ay nasa panig ng nakikita, mga kalahating metro ang layo.
  3. I-tap ang camera button para i-freeze ang imahe at paganahin ang mga kontrol sa alignment.
  4. I-drag ang iyong daliri sa imahe para ma-adjust ang alignment pataas at pababa. Gamitin ang slider para ikutin ito.
  5. Kung ang bagay na ginagamit mo para pagpantayin ang mga imahe ay hindi kalahating metro ang layo, i-set ang layo gamit ang ibabang slider.
  6. I-tap ang I-SAVE para i-apply ang iyong mga pagbabago.

I-revert

  • I-tap ang I-REVERT para i-load ang alignment sa paktorya.

Kanselahin

  • I-tap ang KANSELAHIN para um-exit sa Utility ng Pagre-align ng Imahe na hindi mag-save.
