Sundin ang pamamaraang ito:

  1. I-tap ang button na MODE sa ibaba ng screen para ma-display ang menu ng tagapili ng mode.
  2. Gawin ang isa sa sumusunod:
    • I-tap ang icon ng para makapunta sa mode ng Litrato.
    • I-tap ang icon ng para makapunta sa Video mode.
    • I-tap ang icon ng para makapunta sa mode ng Live Stream.
    • I-tap ang icon ng para makapunta sa mode ng Time-Lapse.
