Pinapahintulutan ka ng Isotherm feature na tukuyin ang partikular na hanay ng temperatura gamit ang highlight na kulay.

  • Para mapagana ang isotherm, i-tap ang toggle button kasunod ng IR Scale.
  • Magde-default ang hanay ng temperatura sa isang hanay sa kalagitnaan ng hanay ng temperatura ng eksena.
  • Ang bawat palette ay may 2 rekumendadong kulay para gamitin ng isotherm, ngunit pwedeng magamit ang ibang mga kulay. I-tap ang kulay na gusto para sa isotherm.
  • Ang hanay ng temperatura ay maaaring itakda sa pamamagitan ng pag-tap sa mga text field ng min at max na temperatura
