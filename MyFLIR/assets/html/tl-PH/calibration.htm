Bilang default, ang imahe ay awtomatikong naka-calibrate para masiguro ang optimized na imahe at mapabuti ang pagkatumpak ng temperatura. Habang isinasagawa ang calibration, naka-freeze ang imahe at ang button ay gagalaw para tukuyin ang progreso.

Maaari kang magpalit-lipat sa pagitan ng awtomatiko at mano-manong mga mode ng pag-adjust. Sa mode na mano-mano, kailangan mong mano-manong i-adjust ang imahe.

  • Para mano-manong i-calibrate ang imahe, i-tap ang button.
  • Para lumipat sa pagitan ng awtomatiko o mano-manong mode, i-tap ang button. Pagkatapos ay i-tap ang icon at i-toggle ang setting sa ilalim ng Automatic calibration.
