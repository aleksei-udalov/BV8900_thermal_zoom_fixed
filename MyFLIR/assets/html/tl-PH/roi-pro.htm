Ang feature ng pagsukat ng RNI (Rehiyon Ng Interes) ay available sa kaparehong paraan ng kapwa I-edit at Live mode. Para sa bawat RNI, ang max/min/average na temperatura sa RNI ay maaaring maipakita, kasama ang mga marker sa kanilang lokasyon.

Mayroong limang uri ng RNI na magagamit â€“ Spot Meter, Rektanggulo (Buong Frame at Maigagalaw), Bilog, at Linya. Ang pagdiin at pag-hold isang RNI (Rehiyon Ng Interes) ay naglalabas sa mga opsyon na burahin ang RNI (Rehiyon Ng Interes) o para ipakita ng minimum at maximum na mga temperatura.

I-tap ang button sa itaas na toolbar at gawin ang isa sa sumusunod:

Spotmeter

  • I-tap ang button para gumawa ng indibidwal na Spotmeter.
  • I-drag ang nalikhang Spotmeter icon sa screen para sukatin ang temperatura sa indibidwal na Spotmeter. Maaari kang magdagdag ng hanggang limang spot meter sa isang imahe.

Buong Frame

  • I-tap ang button para ipakita ang max/min/average na temperatura para sa buong senaryo.

Rektanggulo

  • I-tap ang mas maliit na button ng para ma-display ang max/min/average na temperatura para sa isang parisukat na bahagi ng frame.
  • Ang Rektanggulo ay maaaring palitan ng sukat sa pamamagitan ng paghila mula sa alinmang sulok, at naigagalaw sa pamamagitan ng paghila sa gitna nito.

Bilog

  • I-tap ang button ng para ma-display ang max/min/average na temperatura para sa parihabang area ng frame.
  • Ang bilog ay maaaring mapalitan ang sukat sa pamamagitan ng paghila mula sa gilid, at igalaw sa pamamagitan ng paghila sa gitna nito.

Linya

  • I-tap ang icon ng para maipakita ang max/min/average na temperatura sa isang linya sa frame.
  • Para magpalit sa pagitan ng patayo at pahiga, i-tap nang dalawang beses ang linya.
  • Maaaring igalaw ang linya sa pamamagitan ng paghila.
