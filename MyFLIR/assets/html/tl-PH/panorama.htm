Para sa mga phone na sumusuporta sa accelerometer at gyroscope, maaari mong gamitin ang panorama functionality.

Makakalikha ka ng imahe ng panorama sa pamamagitan ng pag-pan sa kamera mula kaliwa pakanan o mula kanan pakaliwa.

Sundin ang pamamaraang ito:

  1. I-tap ang button na MODE sa ibaba ng screen para ma-display ang menu ng tagapili ng mode.
  2. I-tap ang icon para mag-Panorama mode.
  3. (Opsyonal na hakbang.) I-tap ang icon at pumili ng naiibang palette.
  4. I-tap ang button para umpisahan ang panning sequence.
  5. Hawakan nang steady ang kamera at i-pan mula kaliwa papuntang kanan o mula kanan papuntang kaliwa. Awtomatikong madedetek ng app ang direksyon at magdi-display ng gumagabay na graphics.

  6. I-tap ang button para mahinto ang sequence. Kung matagumpay na malikha ang panorama, awtomatiko itong masi-save,
