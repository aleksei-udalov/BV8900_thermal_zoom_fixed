Ang feature ng pagsukat ng RNI (Rehiyon Ng Interes) ay available sa kaparehong paraan ng kapwa I-edit at Live mode. Para sa bawat RNI (Rehiyon Ng Interes), ipinapakita ang RNI ang max/min/katamtaman na temperatura.

Ang pagdiin at pag-holdn isang RNI (Rehiyon Ng Interes) ay naglalabas sa mga opsyon na burahin ang RNI (Rehiyon Ng Interes) o para ipakita ng minimum at maximum na mga temperatura.

I-tap ang button sa itaas na toolbar at gawin ang isa sa sumusunod:

Spotmeter RNI (Rehiyon ng Interes)

  • I-tap ang button para gumawa ng indibiduwal na Spotmeter.
  • I-drag ang nalikhang Spotmeter icon sa screen para sukatin ang temperatura sa indibidwal na Spotmeter. Maaari kang magdagdag ng hanggang tatlong spot meter sa isang imahe.
  • Idiin at -ihold ang mga indibidwal na icon ng Spotmeter para sa mga opsyon na burahin ang RNI (Rehiyon ng Interes)

Buong Frame na RNI (Rehiyon ng Interes)

  • I-tap ang button para ipakita ang max/min/average na temperatura para sa buong senaryo.
  • Idiin at i-hold saanman sa eksena para sa ilabas ang mga opsyon na burahin ang RNI (Rehiyon ng Interes) o para ipakita ang minimum at maximum na mga temperatura.

Gitnang Kwadradong RNI (Rehiyon ng Interes)

  • I-tap ang mas maliit na icon para ipakita ang max/min/average na temperatura pata sa gitna ng frame.
  • Maaaring baguhin ang laki ng parisukat sa pamamagitan ng paghatak mula sa anumang kanto, pero mananatili itong nakagitna.
  • Idiin at i-hold sa gitna ng frame para ilabas ang mga opsyon na burahin ang RNI (Rehiyon ng Interes) o para ipakita ang minimum at maximum na mga temperatura.
