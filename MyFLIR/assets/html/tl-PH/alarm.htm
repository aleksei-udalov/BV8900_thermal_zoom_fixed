Nagpapagana sa mga alarma na napapasimulan ng temperatura sa pamamagitan ng pagpasok sa isotherm mode at pag-click sa button ng alarma sa menu ng opsyon ng isotherm. Ang mga alarma ay may tatlong opsyon para sa mga event na napapasimulan ng temperatura na naipapakita kapag pinindot ang button ng alarma:

  • Kumuha ng litrato
  • Mag-vibrate
  • Abiso ng push
Ang oras ng paghihintay sa pagitan ng bawat sunod-sunod na alarma ay 5 segundo. Ang button ng alarma ay ipapakita na kapag aktibo ang alarma.
