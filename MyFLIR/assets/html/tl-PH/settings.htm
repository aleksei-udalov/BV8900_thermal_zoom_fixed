Maaari mong palitan ang iba't ibang setting kaugnay ng kilos ng MyFLIR.

Sundin ang pamamaraang ito:

  1. I-tap ang icon para ma-display ang mga tool at menu ng mga setting.
  2. I-tap ang icon para ma-display ang Settings menu.
  3. Sa Settings menu, maaari mong gawin ang sumusunod:
    • Tingnan ang mga file ng tulong at ang impormasyon tungkol sa MyFLIR.
    • Emissivity: Para mabago ang emissivity, i-tap ang Emissivity at piliin ang isa sa sumusunod:
      • Matte (Nirerekumenda).
      • Medyo-Matte.
      • Medyo-Makintab.
      • Makintab (Hindi Nirerekumenda).
    • Temperature unit: Para mapalitan ang unit ng temperatura, i-tap ang °C o °F.
    • I-save ang Lokasyon: Para ma-save ang lokasyon ng GPS sa imahe, i-swipe ang slider sa On.
    • Awtomatikong pag-adjust: Para awtomatikong i-adjust ang mga imahe, i-swipe ang slider sa On.

Ano ang emissivity?

Ang emissivity ay ang kakayahan ng materyal na magpalabas ng infrared light. Para makuha ang tumpak na sukat, kailangang itugma ng emissivity ang material na iyong tinitingnan, Para sa karamihan sa mga material, ang default na setting ng Matte ay isang mahusay na pagtantya,

Ang mga material na makintab, tulad ng mga metal, ay maaaring magbigay ng hindi wastong sukat ng temperatura.
