Pinagsasama ng MSX ang mga balangkas ng mga bagay mula sa nakikitang camera gamit ang thermal na imahe para mas mapaganda ang mga detalye.

Ang lakas ng mga balangkas ng MSX ay pwedeng ma-adjust gamit ang alpha slider bar kapag ang camera ay nasa MSX blending mode

Pwedeng gamitin ang mga single input camera mode sa pamamagitan ng pagpindot sa thermal at nakikitang button habang nasa menu ng mode ng paghahalo o sa edit mode ng gallery. Ang kapwa nakikita at thermal na mga imahe ay makukuha habang nasa single camera mode. Ang mga setting ng anumang nakuhang litrato ay pwedeng ibalik sa ninanais na input mode habang tinitingnan ang mga imahe sa gallery.

Sa MSX mode, pwede kang kumuha ng close-up na litrato/video ng mga bagay sa layo na 0.3–1 m (12–40 in. ).

Sundin ang pamamaraang ito:

  1. Iposisyon ang camera 0.3–1 m (12–40 in.) mula sa bagay.
  2. I-tap ang button sa itaas na toolbar para maipakita ang fusion mode selection drop-down menu.
  3. Gamitin ang slider para i-adjust ang mga balangkas hanggang hanay ang mga ito sa mga dulo ng infrared na mga bagay.
  4. I-tap ang button ng pag-capture button.
