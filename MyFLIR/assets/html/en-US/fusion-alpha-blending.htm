MSX blends outlines of objects from the visible camera with the the thermal image to enhance details.

Alpha blending mode can be enabled using the button. This option displays an overlay of the thermal and visible camera video without MSX outlines. The strength of each input can be adjusted by moving the alpha slider bar.

The MSX outlines strength can be adjusted using the alpha slider bar when the camera is in MSX blending mode

Single input camera modes are available by pressing the thermal and visible button while in the blending mode menu or the gallery edit mode. Both visible and thermal images will be captured while in the single camera mode. Settings for any capture can be changed back to the desired input mode while viewing images in the gallery.

In MSX mode, you can take a close-up photo/video of objects at a distance of 0.3–1 m (12–40 in.).

Follow this procedure:

  1. Position the camera 0.3–1 m (12–40 in.) from the object.
  2. Tap the button in the top toolbar to display the fusion mode selection drop-down menu.
  3. Use the slider to adjust the outlines until they line up with the edges of the infrared object.
  4. Tap the capture button.
