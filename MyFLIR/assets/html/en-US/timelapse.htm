You can create a time-lapse video by letting MyFLIR capture several images in a set time interval and then creating a video clip based on these images.

Follow this procedure:

  1. Tap the MODE button in the bottom of the screen to display the mode selector menu.
  2. Tap the icon to go to TIME-LAPSE mode.
  3. (Optional step.) Tap the icon and select a different palette.
  4. Tap the button to open the Time-Lapse settings menu.
  5. Define the settings. The settings that can be defined are Initial delay (seconds), Frame Interval (seconds), Playback Rate (frames per second), and Show Elapsed Time.
  6. Tap the button to start the recording.
  7. Tap the button to stop the recording.
