Hinweis: Dieser Abschnitt trifft nur auf den Foto- und Videomodus zu.

Sie können MyFLIR so einrichten, dass ein Bild nach einem bestimmten Zeitintervall gespeichert wird.

Folgen Sie den Anweisungen:

  1. Tippen Sie auf das Symbol
  2. Tippen Sie auf das Symbol .
  3. Tippen Sie erneut, um das gewünschte Zeitintervall (drei bis zehn Sekunden) einzustellen.
  4. Tippen Sie auf die Schaltfläche oder (im Foto- bzw. Videomodus), um den Timer zu starten.
