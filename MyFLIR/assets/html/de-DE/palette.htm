Sie können die Farbpalette, die die Kamera verwendet um verschiedene Temperaturen anzuzeigen, ändern. Eine andere Palette vereinfacht es möglicherweise ein Bild zu verändern.

Folgen Sie den Anweisungen:

  1. Tippen Sie auf das Symbol , um das Palettenmenü anzuzeigen.
  2. Tippen Sie auf die gewünschte Palette.
