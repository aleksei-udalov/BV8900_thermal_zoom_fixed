Wenn sich die Ausrichtung der Wärmebildkamera und der eigentlichen Kamera ändert, können Sie manuelle Anpassungen vornehmen.

Verfahren zur Neuausrichtung

  1. Um die Ausrichtung festzulegen, tippen Sie auf das Symbol . Tippen Sie dann auf das Symbol und wählen Sie das Dienstprogramm zur Bild-Neuausrichtung aus.
  2. Platzieren Sie ein Objekt wie z.B. Ihre Hand in das Sichtfeld, etwa einen halben Meter entfernt.
  3. Tippen Sie auf die Kamerataste, um das Bild einzufrieren und die Ausrichtungssteuerung zu aktivieren.
  4. Streichen Sie zur Anpassung der Ausrichtung mit Ihrem Finger auf dem Bild abwärts und aufwärts. Verwenden Sie den Schieberegler, um die Ausrichtung zu drehen.
  5. Wenn das Objekt, mit dem Sie die Bilder ausrichten möchten, nicht einen halben Meter entfernt ist, stellen Sie die Entfernung mit dem unteren Schieberegler ein.
  6. Tippen Sie zum übernehmen der Änderungen auf SPEICHERN.

Zurücksetzen

  • Tippen Sie zum Laden der werksseitigen Ausrichtung auf ZURÜCKSETZEN.

Abbrechen

  • Tippen Sie zum Verlassen des Bildneuausrichtungs-Tools ohne Speicherung auf Abbrechen.
