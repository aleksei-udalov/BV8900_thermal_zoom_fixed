In allen Kameramodi können Sie Nahaufnahmen von Objekten in einer Entfernung von 0,3-1m machen (12–40 Zoll ).

Folgen Sie den Anweisungen:

  1. Positionieren Sie das kamera 0,3-1m (12-40 Zoll) vom Objekt entfernt.
  2. Tippen Sie auf die Schaltfläche in der oberen Symbolleiste, um das Dropdown-Menü für den Fusionsmodus anzuzeigen.
  3. Verwenden Sie den Schieberegler, um die Umrisse anzupassen, bis sie mit den Kanten des Infrarotobjekts ausgerichtet sind.
  4. Tippen Sie auf die Aufnahmetaste.
