Når du lagrer et bilde, blir det lagret i MyFLIRs bildearkiv. Du kan når som helst åpne og legge til en spotmåler, endre paletten, osv.

Trykk på miniatyrbildet nederst på høyre hjørne for å åpne det siste bildet som ble tatt.

Bildeforhåndsvisning

I Bilde-visningen kan du gjøre følgende:

  • Trykk på ikonet for å gå til Bibliotekvisning.
  • Sveip til venstre eller høyre for å gå til forrige/neste bilde/video.
  • (Kun bilder.) Sveip opp/ned for å bytte mellom termiske og visuelle bilder.
  • (Kun bilder.) Trykk på ikonet for å se hvor du kan legge til en spotmåler og endre paletten.
  • (Kun bilder.) Trykk på ikonet for å se informasjon om bildet og eventuelt GPS-plasseringen hvor bildet var tatt.
  • Trykk på ikonet for å dele bildet/videoen på f.eks. e-post, Facebook, eller Twitter.
  • Trykk på ikonet for å dele bildet/videoen.

Biblioteksvisning

I Bibliotek-visning kan du gjøre følgende:

  • Trykk på ikonet for å gå til Bildevisning.
  • Trykk på et ikon for å se hele bildet/videoen.
  • Trykk og hold på miniatyrbildet for å gå til Valgmodus, hvor du kan gjøre følgende:
    • Trykk på miniatyrbildene for å markere / fjerne markering bilder/videoer.
    • Trykk på ikonet for å markere alle bilder/videoer.
    • Trykk på ikonet for å dele de markerte bildene/videoene på e-post, Facebook, eller Twitter.
    • Trykk på ikonet for å slette de markerte bildene/videoene.
