É possível criar um vídeo de lapso de tempo possibilitando que o MyFLIR capture várias imagens em um intervalo definido e, em seguida, criando um clipe de vídeo com base nas imagens.

Siga este procedimento:

  1. Toque no botão MODO na parte inferior da tela para exibir o menu do seletor de modos.
  2. Toque no ícone para acessar o modo LAPSO DE TEMPO.
  3. (Passo opcional.) Toque no ícone e selecione uma paleta diferente.
  4. Toque no botão para abrir o menu Configurações de lapso de tempo.
  5. Defina as configurações. As configurações que podem ser definidas são Atraso inicial (segundos), Intervalo dos quadros (segundos), Taxa de reprodução (quadros por segundo) e Exibir tempo decorrido.
  6. Toque no botão para começar a gravação.
  7. Toque no botão para parar a gravação.
