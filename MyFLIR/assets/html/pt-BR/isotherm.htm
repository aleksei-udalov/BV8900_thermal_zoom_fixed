O recurso Isotérmico permite que você indique um intervalo de temperatura específico com uma cor de destaque.

  • Para ativar o isotérmico, toque no botão ao lado da escala IV.
  • O intervalo de temperatura será padronizada para uma faixa no meio do intervalo de temperatura da cena.
  • Cada paleta possui 2 cores para uso recomendadas pelo isotérmico, mas outras cores também estão disponíveis. Toque na cor desejada para o isotérmico.
  • O intervalo de temperatura pode ser definida tocando nos campos de texto de temperatura mínima e máxima
