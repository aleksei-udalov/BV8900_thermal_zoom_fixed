Para telefones com suporte para acelerômetro e giroscópio, é possível usar a funcionalidade de panorama.

É possível criar uma imagem de panorama movendo a câmera da direita para a esquerda ou da esquerda para a direita.

Siga este procedimento:

  1. Toque no botão MODO na parte inferior da tela para exibir o menu do seletor de modo.
  2. Toque no ícone para acessar o modo Panorama.
  3. (Passo opcional.) Toque no ícone e selecione uma paleta diferente.
  4. Toque no botão para iniciar a sequência de movimento.
  5. Segure firmemente a Câmera e mova-a da esquerda para a direita ou da direita para a esquerda. O aplicativo automaticamente detecta o sentido e exibe gráficos de orientação.

  6. Toque no botão para parar a sequência. Se o panorama for criado com sucesso, ele será salvo automaticamente.
