Se o alinhamento das câmeras térmica e visível muda, é possível fazer ajustes manuais.

Procedimento de realinhamento

  1. Para configurar o alinhamento, toque no ícone . Em seguida, toque no ícone e selecione o Utilitário de realinhamento de imagens.
  2. Coloque um objeto, como a mão, no campo de visão a cerca de meio metro de distância.
  3. Toque no botão da câmera para congelar a imagem e ativar os controles de alinhamento.
  4. Arraste o dedo ao longo da imagem para ajustar o alinhamento para cima e para baixo. Use o controle deslizante para girá-la.
  5. Se o objeto que você está usando para alinhar as imagens não está a meio metro de distância, defina a distância com o controle deslizante inferior.
  6. Toque em SALVAR para aplicar as alterações.

Reverter

  • Toque em REVERTER para carregar o alinhamento de fábrica.

Cancelar

  • Toque em CANCELAR para sair do Utilitário de realinhamento de imagens sem salvar.
