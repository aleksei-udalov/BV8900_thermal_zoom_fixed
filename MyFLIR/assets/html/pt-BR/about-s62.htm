Obrigado por escolher o Cat® S62 Pro; com MyFLIR Pro!

  • Com a capacidade de ver o que o olho nu não consegue, o MyFLIR Pro, alimentado pelo núcleo de imagens térmicas Lepton 3.5, fornece uma imagem visual das diferenças mínimas de temperatura, dando aos usuários o poder de ver no escuro, observar fontes de calor invisíveis, comparar temperaturas relativas e ver através da fumaça.
  • Criado para aprimorar a experiência do dispositivo da câmera, o MyFLIR Pro fornece uma interface de usuário simples e intuitiva com uma variedade de funções aprimoradas que o tornam fácil de personalizar, coletar, modificar e compartilhar imagens térmicas úteis e bonitas. O aplicativo MyFLIR Pro apresenta uma caixa de ferramentas com novos recursos que permitem análise detalhada de transmissão ou imagens térmicas capturadas.
  • Alguns desses novos recursos incluem Isotermas (faixas de temperatura destacadas), Alarmes Acionados por Temperatura e novas opções de Modo de Mesclagem para a captura de dados térmicos. Tudo isso é emparelhado com o VividIR, que melhora os detalhes significativamente comparados à imagem térmica padrão do MyFLIR.
