É possível alterar uma variedade de configurações relacionadas ao comportamento do MyFLIR.

Siga este procedimento:

  1. Toque no ícone para exibir o menu de ferramentas e configurações.
  2. Toque no ícone para exibir o menu Configurações.
  3. No menu Configurações, é possível fazer o seguinte:
    • Exibir arquivos de ajuda e informações sobre o MyFLIR.
    • Emissividade: Para alterar a emissividade, toque em Emissividade e selecione um dos seguintes:
      • Matte (recomendado).
      • Semimatte.
      • Semibrilhante.
      • Brilhante (não recomendado).
    • Unidade de temperatura: Para alterar a unidade de temperatura, toque em °C ou °F.
    • Salvar local: Para salvar o local do GPS na imagem, deslize o controle deslizante para Ligado.
    • Calibragem automática: Para calibrar as imagens automaticamente, deslize o controle deslizante para Ligado.

O que é emissividade?

Emissividade é a capacidade de um material de emitir luz infravermelha. Para obter uma medição precisa, a emissividade deve corresponder ao material sendo exibido. Para a maioria dos materiais, a configuração padrão Matte é uma boa aproximação.

Materiais brilhantes, como metais, podem fornecer medições de temperatura incorretas.
