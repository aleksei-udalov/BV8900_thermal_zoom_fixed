Observação: Esta seção se aplica somente aos modos Foto e Vídeo.

É possível confirmar o MyFLIR para que salve uma imagem depois de intervalo especificado.

Siga este procedimento:

  1. Toque no ícone para exibir o menu de ferramentas e configurações.
  2. Toque no ícone .
  3. Toque novamente para selecionar o intervalo desejado (3 ou 10 segundos).
  4. Toque no botão ou (no modo Foto ou Vídeo, respectivamente) para iniciar o cronômetro.
