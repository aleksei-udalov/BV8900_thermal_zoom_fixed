Ao salvar uma imagem, ela é salva na galeria de imagem do MyFLIR. É possível, a qualquer momento, abrir uma imagem e adicionar um medidor de ponto, alterar a paleta etc.

Para abrir a última imagem obtida, toque na miniatura da imagem no canto inferior direito da tela.

Vista de Imagem

Na vista de Imagem, é possível fazer o seguinte:

  • Toque no ícone para acessar a vista de Biblioteca.
  • Deslize para a direita/esquerda para acessar o vídeo/imagem anterior/posterior.
  • (Somente imagens.) Deslize para cima/para baixo para alternar entre imagens térmicas e visuais.
  • (Somente imagens.) Toque no ícone para exibir uma vista em que é possível adicionar um medidor de ponto e alterar a paleta.
  • (Somente imagens.) Toque no ícone para exibir informações sobre a imagem e, opcionalmente, o local do GPS onde ela foi obtida.
  • Toque no ícone para compartilhar a imagem/vídeo usando, por exemplo, e-mail, Facebook ou Twitter.
  • Toque no ícone para excluir a imagem/vídeo.

Vista de Biblioteca

Na vista de Biblioteca, é possível fazer o seguinte:

  • Toque no ícone para acessar a vista de Imagem.
  • Toque em uma miniatura para ver a imagem/vídeo completo.
  • Mantenha pressionada uma miniatura para acessar o modo Seleção, onde é possível fazer o seguinte:
    • Toque nas miniaturas para selecionar/remover a seleção de imagens/vídeos.
    • Toque no ícone para selecionar todas as imagens/vídeos.
    • Toque no ícone para compartilhar as imagens/vídeos selecionados usando, por exemplo, e-mail, Facebook ou Twitter.
    • Toque no ícone para excluir as imagens/vídeos selecionados.
