Ative os alarmes acionados por temperatura entrando no modo isotérmico e clicando no botão de alarme no menu de opções isotérmicas. Os alarmes têm três opções para eventos acionados por temperatura que são exibidos quando o botão de alarme é pressionado:

  • Capturar foto
  • Vibrar
  • Notificação Push
O período de espera entre cada alarme sucessivo é de 5 segundos. O botão de alarme será exibido como quando o alarme estiver ativo.
