O recurso de medição de região de interesse (ROI) está disponível da mesma forma nos modos Editar e Ao Vivo. Para cada ROI, a temperatura máx/min/média no ROI pode ser exibida juntamente com ícones de sua localização.

Existem cinco tipos de ROI disponíveis – Medidor de Ponto, Retângulo (Quadro Completo e Móvel), Círculo, e Linha. Manter pressionada uma ROI abre opções para excluí-la ou para exibir as temperaturas mínima e máxima.

Toque no botão na barra de ferramentas superior e faça um dos seguintes:

Medidor de Ponto

  • Toue no botão para criar um Medidor de ponto individual.
  • Arraste o ícone do Medidor de ponto gerado pela tela para medir a temperatura de um Medidor de ponto individual. Você pode adicionar até cinco medidores de ponto a uma imagem.

Quadro Completo

  • Toque no botão para exibir a temperatura máxima/mínima/média de toda a cena.

Retângulo

  • Toque no botão menor para exibir a temperatura máx/min/média para uma área retangular do quadro.
  • O retângulo pode ser redimensionado arrastando de qualquer canto e movido arrastando para o centro.

Círculo

  • Toque no botão para exibir a temperatura máx/min/média para uma área circular do quadro.
  • O círculo pode ser redimensionado arrastando da borda e movido arrastando para o centro.

Linha

  • Toque no ícone para exibir a temperatura máx/min/média ao longo de uma linha no quadro.
  • Para alternar entre vertical e horizontal, toque duas vezes na linha.
  • Arraste a linha para movê-la.
