Vous pouvez modifier différents paramètres du fonctionnement de MyFLIR.

Suivez cette procédure :

  1. Touchez l'icône pour afficher les outils et le menu paramètres.
  2. Touchez l'icône pour afficher le menu Paramètres.
  3. Dans le menu Paramètres, vous pouvez procéder comme suit :
    • Afficher les fichiers d'aide et les informations sur MyFLIR.
    • Émissivité : pour modifier l'émissivité, touchez Émissivité et sélectionnez l'une des options suivantes :
      • Mat (recommandé).
      • Semi-Mat.
      • Semi-brillant.
      • Brillant (non recommandé).
    • Unité de température : pour modifier l'unité de température, touchez °C ou °F.
    • Enregistrer la position : pour enregistrer la localisation GPS dans l'image, mettez l'interrupteur sur On.
    • Calibration automatique : pour calibrer les images automatiquement, mettez l'interrupteur sur On.

Qu'est-ce que l'émissivité ?

L'émissivité est la capacité d'un matériau à émettre des rayons infrarouges. Pour obtenir une mesure exacte, l'émissivité doit correspondre au matériau observé. Pour la plupart des matériaux, le réglage Mat par défaut constitue une bonne approximation.

Les matériaux brillants, tels que les métaux, peuvent donner des mesures de température incorrectes.
