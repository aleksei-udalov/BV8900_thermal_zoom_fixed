Dans tous les modes caméra, vous pouvez enregistrer une photo ou une vidéo en gros plan d'un objet à une distance de 0,3 à 1 m (12 à 40 pouces ).

Suivez cette procédure :

  1. Placez la caméra de 0,3 à 1 m (12 à 40 pouces) de distance de l'objet.
  2. Touchez le bouton dans la barre d'outils supérieure pour afficher le menu déroulant de sélection du mode fusion.
  3. Utilisez le curseur pour ajuster les contours jusqu'à ce qu'ils soient alignés avec les bords de l'image infrarouge.
  4. Touchez le bouton d'enregistrement.
