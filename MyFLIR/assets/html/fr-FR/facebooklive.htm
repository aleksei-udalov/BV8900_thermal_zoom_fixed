Suivez cette procédure :

  1. Touchez le bouton MODE en bas de l'écran pour afficher le menu de sélection de mode.
  2. Touchez l'icône pour passer en mode Facebook Live.
  3. Connectez-vous sur votre compte Facebook et autorisez MyFLIR à publier des vidéos en direct.
  4. Orientez la caméra vers l'objet qui vous intéresse.
  5. Utilisez le menu déroulant de sélection du public à côté de l'icône d'obturateur pour choisir qui peut voir votre vidéo :
    • public :
    • amis :
  6. Touchez le bouton pour démarrer la diffusion en direct.
  7. Touchez le bouton pour mettre fin à la diffusion en direct.
