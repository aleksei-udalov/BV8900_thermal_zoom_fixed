Suivez cette procédure :

  1. Touchez le bouton MODE en bas de l'écran pour afficher le menu de sélection de mode.
  2. Effectuez l'une des opérations suivantes.
    • Touchez l'icône pour passer en mode Photo.
    • Touchez l'icône pour passer en mode Vidéo.
    • Touchez l'icône pour passer en mode Live.
    • Touchez l'icône pour passer en mode Time-lapse.
