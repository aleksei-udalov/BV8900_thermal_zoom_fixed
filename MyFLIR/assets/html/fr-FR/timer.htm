Remarque : cette section s'applique aux modes photo et vidéo.

Vous pouvez configurer MyFLIR pour enregistrer une image après un délai défini.

Suivez cette procédure :

  1. Touchez l'icône pour afficher les outils et le menu paramètres.
  2. Touchez l'icône .
  3. Touchez à nouveau pour sélectionner le délai que vous souhaitez (de 3 à 10 secondes).
  4. Touchez le bouton ou (en mode Photo ou Vidéo, respectivement) pour lancer le retardateur.
