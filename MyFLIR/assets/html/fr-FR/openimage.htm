Les images sont enregistrées dans la galerie d'images de MyFLIR. Vous pouvez ouvrir une image à tout moment pour ajouter une mesure, modifier la palette, etc.

Pour ouvrir la dernière image enregistrée, touchez la vignette de l'image dans le coin en bas à droite de l'écran.

Affichage d'une image

Dans l'affichage d'image, vous pouvez...

  • Toucher l'icône pour aller dans la bibliothèque.
  • Balayer vers la gauche ou la droite pour aller à la photo ou vidéo suivante ou précédente.
  • (Images uniquement) Balayer vers le haut ou le bas pour passer de l'image thermique à l'image « visible ».
  • (Images uniquement) Toucher l'icône pour afficher un mode où vous pouvez ajouter une mesure avec un spotmètre et modifier la palette.
  • (Images uniquement) Toucher l'icône pour afficher les informations de l'image et éventuellement la localisation GPS du lieu d'enregistrement de l'image.
  • Toucher l'icône pour partager la photo ou vidéo par e-mail ou sur Twitter ou Facebook par exemple.
  • Toucher l'icône pour supprimer l'image ou la vidéo.

Bibliothèque

Dans la bibliothèque, vous pouvez...

  • Toucher l'icône pour aller dans l'affichage d'image.
  • Toucher une vignette pour afficher une photo ou vidéo.
  • Maintenir l'appui sur une vignette pour aller dans le mode Sélection, où vous pouvez :
    • toucher les vignettes pour sélectionner/désélectionner les images ou vidéos,
    • toucher l'icône pour sélectionner/désélectionner toutes les images ou vidéos,
    • toucher l'icône pour partager les photos ou vidéos sélectionnées par e-mail ou sur Twitter ou Facebook par exemple,
    • toucher l'icône pour supprimer les images ou vidéos sélectionnées.
