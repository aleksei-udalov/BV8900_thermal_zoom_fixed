Créez une vidéo accélérée en laissant MyFLIR effectuer des prises de vues à intervalle régulier et créer une séquence vidéo à partir de ces images.

Suivez cette procédure :

  1. Touchez le bouton MODE en bas de l'écran pour afficher le menu de sélection de mode.
  2. Touchez l'icône pour passer en mode TIME-LAPSE.
  3. (Étape facultative) Touchez l'icône et sélectionnez une autre palette.
  4. Touchez le bouton pour ouvrir le menu des paramètres de Time-Lapse.
  5. Configurez les paramètres. Les paramètres pouvant être modifiés sont Retard initial (en secondes), Intervalle de prise de vues (en secondes), Vitesse de lecture (en images/s) et Afficher le temps écoulé.
  6. Touchez le bouton pour lancer l'enregistrement.
  7. Touchez le bouton pour mettre fin à l'enregistrement.
