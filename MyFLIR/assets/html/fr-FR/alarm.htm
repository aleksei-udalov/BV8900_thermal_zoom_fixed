Utilisez des alarmes déclenchées par une température : passez en mode isotherme et cliquez sur le bouton d'alarme dans le menu des options d'isotherme. Les alarmes ont trois options d'événements déclenchés par une température, qui s'affichent quand vous appuyez sur le bouton de l'alarme :

  • Prendre une photo
  • Vibrer
  • Recevoir une notification
. La période d'attente entre les alarmes successives est de 5 secondes. Quand l'alarme est active, le bouton d'alarme prend cette forme : .
