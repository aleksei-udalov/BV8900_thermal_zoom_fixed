Suivez cette procédure :

  1. Touchez le bouton MODE en bas de l'écran pour afficher le menu de sélection de mode.
  2. Touchez l'icône pour passer en mode vidéo.
  3. Orientez la caméra vers l'objet qui vous intéresse.
  4. Touchez le bouton pour lancer l'enregistrement.
  5. Touchez le bouton pour mettre fin à l'enregistrement.
