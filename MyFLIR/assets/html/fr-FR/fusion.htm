MSX fusionne les contours des objets de la caméra « visible » avec l'image thermique pour renforcer les détails.

L'intensité des contours MSX peut être ajustée à l'aide du curseur alpha lorsque la caméra est en mode de fusion MSX.

Les modes de caméra à entrée unique sont accessibles via le bouton thermique et le bouton visible dans le menu du mode de fusion ou dans le mode Modifier de la galerie. En mode caméra à entrée unique, les images visible et thermique sont toutes deux enregistrées. Les paramètres de n'importe quelle image peuvent être restaurés vers le mode d'entrée voulu lorsque les images sont affichées dans la galerie.

En mode MSX, vous pouvez enregistrer une photo ou vidéo rapprochée d'objets se trouvant à une distance de 0,3 à 1 m (12 à 40 pouces ).

Suivez cette procédure :

  1. placez la caméra de 0,3 à 1 m (12 à 40 pouces) de distance de l'objet.
  2. Touchez le bouton dans la barre d'outils supérieure pour afficher le menu déroulant du mode de fusion.
  3. Utilisez le curseur pour ajuster les contours jusqu'à ce qu'ils s'alignent avec les bords de l'objet infrarouge.
  4. Touchez le bouton d'enregistrement.
