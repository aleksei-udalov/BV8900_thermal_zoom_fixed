La función Isoterma le permite indiciar un rango de temperatura específico con un color de resaltado.

  • Para activar la isoterma, pulse el botón deslizante que hay junto a la escala de infrarrojos.
  • El rango de temperatura se establece de forma predeterminada en el medio del rango de temperatura de la escena.
  • Cada paleta tiene dos colores recomendados para usarlos en la isoterma, pero también hay otros colores disponibles. Pulse el color que desea para la isoterma.
  • El rango de temperatura se puede establecer tocando los campos de texto para la temperatura máxima y mínima.
