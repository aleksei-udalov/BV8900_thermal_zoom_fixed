Puede utilizar la función de imagen panorámica con teléfonos compatibles con el acelerómetro y el giroscopio.

Puede crear una imagen panorámica moviendo la cámara de izquierda a derecha o de derecha a izquierda.

Siga este procedimiento:

  1. Pulse el botón MODO en la parte inferior de la pantalla para mostrar el menú del selector de modo.
  2. Pulse el icono para ir al modo panorámica.
  3. (paso opcional.) Pulse el icono y seleccione una paleta distinta.
  4. Pulse el botón para empezar a realizar la panorámica.
  5. Mantenga la cámara firmemente y muévalo de izquierda a derecha o de derecha a izquierda. La aplicación detecta de forma automática la dirección y muestra unos gráficos de guía.

  6. Pulse el botón para detener la secuencia. Si la imagen panorámica se crea correctamente, se guardará de forma automática.
