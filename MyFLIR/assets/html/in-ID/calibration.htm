Anda dapat beralih di antara mode kalibrasi otomatis dan manual.

Dalam mode otomatis, gambar otomatis dikalibrasi untuk memastikan gambar telah optimal dan untuk meningkatkan akurasi suhu. Dalam mode manual. Anda harus mengkalibrasi gambar secara manual. Sementara kalibrasi sedang berlangsung, gamabr dikaburkan.
