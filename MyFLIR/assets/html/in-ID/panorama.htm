Anda dapat memakai fungsi panorama pada ponsel yang mendukung akselerometer dan giroskop.

Anda dapat membuat gambar panorama dengan menggerakkan kamera dari kiri ke kanan atau dari kanan ke kiri.

Ikuti prosedur ini:

  1. Ketuk tombol MODE di bagian bawah layar untuk menampilkan menu pemilih mode.
  2. Ketuk ikon untuk membuka mode Panorama.
  3. (langkah Opsional.) Ketuk ikon dan pilih palet lainnya.
  4. Ketuk ikon untuk memulai urutan panning.
  5. Pegang agar stabil dan gerakkan dari kiri ke kanan atau dari kanan ke kiri. Aplikasi otomatis mendeteksi arah dan menampilkan grafik pemandu.

  6. Ketuk tombol untuk menghentikan urutan. Jika panorama berhasil dibuat, maka akan otomatis disimpan.
