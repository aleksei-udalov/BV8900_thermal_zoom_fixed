Ikuti prosedur ini:

  1. Untuk menetapkan penyejajaran, ketuk ikon. Lalu ketuk ikon dan pilih Utilitas Penyejajaran Ulang Gambar.
  2. Lakukan salah satu yang berikut:

Penyejajaran Manual

  • Ketuk MANUAL untuk menjeda dan menyesuaikan penyejajaran
  • Tempatkan tangan di bidang tampilan, sekitar setengah meter jauhnya.
  • Seret jari Anda di sepanjang gambar untuk menyesuaikan penyejajaran ke atas dan ke bawah. Gunakan penggeser untuk memutarnya.
  • Ketuk SIMPAN untuk menerapkan perubahan Anda.

Penyejajaran Otomatis

  • Ketuk OTOMATIS untuk menjalankan algoritme penyejajaran otomatis.
  • Tempatkan tangan di bidang tampilan, sekitar setengah meter jauhnya.
  • Setelah penyejajaran otomatis berjalan, Anda dapat melakukan penyejajaran otomatis jika perlu.
  • Ketuk SIMPAN untuk menerapkan perubahan Anda.

Mengembalikan

  • Ketuk KEMBALIKAN untuk memuat penyejajaran baku dari pabrik.

Membatalkan

  • Ketuk BATALKAN untuk keluar dari Utilitas Penyejajaran Ulang Gambar tanpa menyimpan.
