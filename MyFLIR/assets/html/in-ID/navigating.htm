Ikuti prosedur ini:

  1. Ketuk tombol MODE di bagian bawah layar untuk menampilkan menu pemilih mode.
  2. Lakukan salah satu yang berikut:
    • Ketuk ikon untuk membuka mode Foto.
    • Ketuk ikon untuk membuka mode Video.
    • Ketuk ikon untuk membuka mode Panorama.
    • Ketuk ikon untuk membuka mode Kronologis.
