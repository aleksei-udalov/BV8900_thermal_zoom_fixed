Όταν αποθηκεύετε μια εικόνα, αποθηκεύεται στη συλλογή εικόνων του MyFLIR. Μπορείτε να ανοίξετε οποιαδήποτε στιγμή μια εικόνα και να προσθέσετε έναν μετρητή σποτ, να αλλάξετε την παλέτα κλπ.

Για να ανοίξετε την τελευταία ληφθείσα εικόνα, πατήστε στη μικρογραφία της εικόνας στην κάτω δεξιά γωνία της οθόνης.

Προβολή Εικόνας

Στην προβολή Εικόνας, μπορείτε να κάνετε τα εξής:

  • Πατήστε το εικονίδιο για να μεταβείτε στην προβολή Βιβλιοθήκης.
  • Σαρώστε προς τα αριστερά/δεξιά για μετάβαση στην προηγούμενη/επόμενη εικόνα/βίντεο.
  • (Μόνο για εικόνες.) Σαρώστε προς τα πάνω/κάτω για εναλλαγή ανάμεσα σε θερμικές και οπτικές εικόνες.
  • (Μόνο για εικόνες.) Πατήστε το εικονίδιο για να εμφανίσετε μια προβολή όπου μπορείτε να προσθέσετε έναν μετρητή φωτός και να αλλάξετε την παλέτα.
  • (Μόνο για εικόνες.) Πατήστε το εικονίδιο για να εμφανίσετε πληροφορίες σχετικά με την εικόνα και, προαιρετικά, την τοποθεσία GPS όπου πραγματοποιήθηκε η λήψη της εικόνας.
  • Πατήστε το εικονίδιο για να κοινοποιήσετε την εικόνα/το βίντεο μέσω π.χ. e-mail, Facebook ή Twitter.
  • Πατήστε το εικονίδιο για να διαγράψετε την εικόνα/το βίντεο.

Προβολή Βιβλιοθήκης

Στην προβολή Βιβλιοθήκης, μπορείτε να κάνετε τα εξής:

  • Πατήστε το εικονίδιο για να μεταβείτε στην προβολή Εικόνας.
  • Αγγίξτε μια μικρογραφία για να εμφανίσετε την πλήρη εικόνα/βίντεο.
  • Πατήστε παρατεταμένα μία μικρογραφία για να μεταβείτε στη λειτουργία Επιλογής, όπου μπορείτε να:
    • Πατήσετε μικρογραφίες για να επιλέξετε/αποεπιλέξετε εικόνες/βίντεο.
    • Πατήσετε το εικονίδιο για να επιλέξετε όλες τις εικόνες/τα βίντεο.
    • Πατήσετε το εικονίδιο για να κοινοποιήσετε τις επιλεγμένες εικόνες/βίντεο μέσω π.χ.e-mail, Facebook ή Twitter.
    • Πατήσετε το εικονίδιο για να διαγράψετε τις επιλεγμένες εικόνες/βίντεο.
