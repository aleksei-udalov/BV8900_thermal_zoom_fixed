Δημιουργήστε ένα time-lapse βίντεο, επιτρέποντας στην εφαρμογή MyFLIR να πραγματοποιήσει λήψη πολλαπλών φωτογραφιών σε ένα συγκεκριμένο χρονικό διάστημα και, στη συνέχεια, να δημιουργήσει ένα βίντεο με χρήση αυτών των φωτογραφιών.

Ακολουθήστε τα παρακάτω βήματα:

  1. Πατήστε το κουμπί ΛΕΙΤΟΥΡΓΙΑ στο κάτω μέρος της οθόνης για να εμφανιστεί το μενού επιλογής κατάστασης.
  2. Πατήστε το εικονίδιο για μετάβαση στη λειτουργία TIME-LAPSE.
  3. (προαιρετικό βήμα.) Πατήστε το εικονίδιο και επιλέξτε μια διαφορετική παλέτα.
  4. Πατήστε το κουμπί για να ανοίξετε το μενού Ρυθμίσεις Time-Lapse.
  5. Ορίστε τις ρυθμίσεις. Οι ρυθμίσεις που μπορούν να προσαρμοστούν είναι οι εξής: Αρχική καθυστέρηση (δευτερόλεπτα), Διάστημα ανά καρέ (δευτερόλεπτα), Ρυθμός αναπαραγωγής (καρέ ανά δευτερόλεπτο) και Προβολή παρερχόμενου χρόνου.
  6. Πατήστε το κουμπί για να ξεκινήσετε την εγγραφή.
  7. Πατήστε το κουμπί για να σταματήσετε την εγγραφή.
