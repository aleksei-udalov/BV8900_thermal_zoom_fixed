MSX mengadunkan garis kasar objek daripada kamera boleh lihat dengan gambar haba untuk meningkatkan perincian.

Mod pengadunan alfa boleh diaktifkan menggunakan butang . Pilihan ini memaparkan lapisan video kamera haba dan boleh dilihat tanpa garis kasar MSX. Kekuatan setiap input boleh dilaraskan dengan menggerakkan palang gelongsor slaid alfa.

Kekuatan garis kasar MSX boleh dilaraskan menggunakan palang slaid alfa ketika kamera dalam mod adunan MSX

Mod kamera input tunggal tersedia dengan menekan butang haba dan boleh dilihat ketika berada dalam menu mod pengadunan atau mod suntingan galeri. Kedua-dua gambar boleh dilihat dan haba akan dirakam ketika dalam mod kamera tunggal. Tetapan untuk sebarang rakaman boleh dapat diubah kembali kepada mod input yang dikehendaki ketika melihat gambar di galeri.

Dalam mod MSX, anda boleh merakamkan foto/video objek jarak sedekat 0.3-1m (12-40 inci ).

Ikuti prosedur ini:

  1. Letakkan kamera: 0.3-1m (12-40 inci) daripada objek.
  2. Ketik butang di palang alat atas untuk memaparkan menu lungsur jatuh pemilihan mod fusion.
  3. Gunakan lungsur untuk melaraskan garis kasar sehingga ia sejajar dengan pinggir objek inframerah.
  4. Ketik butang rakam.
