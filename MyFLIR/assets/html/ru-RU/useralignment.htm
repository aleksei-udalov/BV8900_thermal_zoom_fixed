Если выравнивание тепловизора и камеры изменилось, вы можете откорректировать его вручную.

Повторное выравнивание

  1. Чтобы настроить выравнивание, коснитесь значка . Затем коснитесь значка и выберите Программа выравнивания изобр.
  2. Поместите руку или другой объект в поле видимости на расстоянии примерно полуметра.
  3. Коснитесь кнопки камеры, чтобы зафиксировать изображение и включите элементы управления выравниванием.
  4. Перетащите палец по изображению, чтобы откорректировать выравнивание вверх и вниз. Воспользуйтесь ползунком, чтобы повернуть его.
  5. Если объект, используемый для выравнивания, находится на другом расстоянии, укажите его с помощью ползунка внизу.
  6. Коснитесь СОХРАНИТЬ, чтобы применить изменения.

Вернуть

  • Коснитесь ВЕРНУТЬ, чтобы восстановить выравнивание по умолчанию.

Отмена

  • Коснитесь кнопки ОТМЕНА, чтобы выйти из программы выравнивания изображений без сохранения.
