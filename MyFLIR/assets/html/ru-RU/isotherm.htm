Функция изотермы дает вам возможность отображать конкретный диапазон температур с помощью выбранного цвета.

  • Чтобы включить изотерму, коснитесь кнопки-переключателя рядом со шкалой ИК.
  • По умолчанию температурный диапазон выбирается в середине температурного диапазона объектов в кадре.
  • Каждая палитра имеет 2 рекомендованных цвета для использования в изотерме, но доступны и другие цвета. Коснитесь цвета для отображения изотермы.
  • Температурный диапазон можно задать, коснувшись текстовых полей с минимальной и максимальной температурой.
